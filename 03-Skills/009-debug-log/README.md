# debug-log

**指令：** `/debug-log`

## 是什麼

讀取 PM2 log 和 Nginx error log，自動判斷錯誤類型並給出解法。
專為 looptw.com 伺服器設計，含常見錯誤的完整排查流程。

## 怎麼用

在 Claude Code 輸入 `/debug-log`，描述症狀：

| 症狀 | 說法 |
|---|---|
| 網頁顯示 502 | 「網站 502 了」 |
| PM2 一直重啟 | 「PM2 一直 restart」 |
| 白畫面 / 404 | 「網站白畫面」 |
| 改了 .env 沒生效 | 「環境變數沒更新」 |
| 推了更新沒變化 | 「更新後網站沒變」 |

Skill 會提供對應的 SSH 指令讓你複製執行，並根據 log 內容判斷根本原因。
